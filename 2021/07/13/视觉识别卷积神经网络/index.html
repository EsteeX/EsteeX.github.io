隐藏层层数即大小

但是不能因此使用小型网络

一般——改变正则化强度，辍学、输入噪声

较小的网络更难用梯度下降等局部方法训练

更大的神经网络包含明显更多的局部最小值，但这些最小值在实际损失方面要好得多

小型网络——在某些情况下，您很幸运并收敛到一个好地方，但在某些情况下，您会陷入其中一个糟糕的最小值

大型网络——最终实现的损失的方差会小得多，（所有的解决方案都一样好），并且更少依赖随机初始化的运气

设置数据和模型

数据预处理

  1. 均值减法

  2. 归一化

    不同的输入特征具有不同的尺度（或单位），那么应用这种预处理才有意义。在图像的情况下，像素的相对比例已经大致相等（范围从 0 到 255），因此执行这个额外的预处理步骤并不是绝对必要的。

  1. PCA 主成分分析

    PCA不能确保可以提高最终的分类效果

    降低维数，能最大限度的保持原有信息，选择样本点投影具有最大方差的方向（ 在信号处理中认为信号具有较大的方差，噪声有较小的方差，信噪比就是信号与噪声的方差比，越大越好。）

  2. Whitening

    原始图像相邻像素值具有高度相关性，所以图像数据信息冗余

    白化作用：

    1，减少特征之间的相关性；2，特征具有相同的方差（协方差阵为1）；

    步骤：

    1.PCA预处理；

    2.白化

    对上面的pca的新坐标X’,每一维的特征做一个标准差归一化处理。

参数初始化

通过适当的数据归一化，可以合理地假设大约一半的权重为正，其中一半为负。

全零初始化 错误——如果神经元的权重初始化为相同，则它们之间没有不对称的来源。

希望权重非常接近于零——初始化为小数——对称破坏

W = 0.01* np.random.randn(D,H)

解决未校准方差问题

用 1/sqrt(n) 校准方差

将每个神经元输出的方差归一化为 1

w = np.random.randn(n) / sqrt(n)，其中n是其输入的数量

稀疏初始化

将所有权重矩阵设置为零，但要打破对称性，每个神经元都随机连接（权重从上面的小高斯采样）到其下方的固定数量的神经元。

初始化偏差

使用 0 偏差初始化，对所有偏差使用小的常数值，例如 0.01

建议是使用 ReLU 单元并使用w = np.random.randn(n) * sqrt(2.0/n)
